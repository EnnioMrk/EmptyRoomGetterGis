# Empty Room Getter

Gets all the empty rooms for a period in the schhol "Gymnasium im Schloss"

## Installing
#### 1. Install Bun
Windows:
```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```

Linux & macOs
```bash
curl -fsSL https://bun.sh/install | bash
```

#### 2. Install dependencies:
```bash
bun install
```

#### Run:
```bash
bun start
```
